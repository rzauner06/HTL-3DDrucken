    Bottom Horns

    Your duct should match the length of your hotend. If you are using Beacon or Klicky, choose the appropriate variant for that ABL method.

    See the Intro Guide's compatibility section for more information and a table of hotends and ducts.

    RevB 34, RevH 40.15 mm

    General Notes

    This is a lightweight version of the stock EVA dual horns.

    Variant Notes

    • The forward mount point is for Rev B, the rear for Rev H.
    • The Beacon kit should include the M3 screws needed to attach the Beacon PCB.
    • This variant does not use an insert to attach the rear inlet to the bottom horns.
    • RevH only: Do not use a FHCS longer than 5mm to attach the rear inlet. It will cause interference.
